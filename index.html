O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem usada para criar páginas web.

O HTML é usado para estruturar o conteúdo na web, como textos, imagens, links, tabelas, vídeos, e muitos outros elementos. Ele permite definir a estrutura de uma página web, informando ao navegador como exibir esses elementos.

  • O HTML não é uma linguagem de programação, mas uma linguagem de marcação, ou seja, ele "marca" partes de um conteúdo com tags (etiquetas) específicas.
  • Por exemplo, quando você escreve <h1> no HTML, você está dizendo ao navegador que aquele texto deve ser exibido como um título principal.

O que são as tags?

As tags são os elementos básicos do HTML e são usadas para estruturar o conteúdo. Elas indicam ao navegador como exibir o conteúdo em uma página web.

Existem diversos tipos de tags, e cada uma tem uma função específica. Aqui estão alguns exemplos:

  1. <p;: Para parágrafos.
  2. <h1> a <h6>: Para títulos (h1 é o maior título, h6 é o menor).
  3. <a>: Para links.

Outras tags de formatação de texto:

  • <strong>: Para deixar o texto em negrito.
  • <em>: Para deixar o texto em itálico.
  • <u>: Para sublinhar o texto.
  • <del>: Para riscar o texto.
  • <mark>: Para destacar o texto com fundo amarelo.
  • <blockquote>: Para citar um trecho em bloco.
  • <sub>: Para texto em subscrito.
  • <sup>: Para texto em sobrescrito.
  • <pre>: Para manter o formato e quebras de linha do texto.

Atributos HTML

Os atributos fornecem informações extras sobre as tags. Um exemplo comum é o atributo href usado em links:

<a href="https://www.exemplo.com">Visite o site</a>

href é o atributo que define o destino do link. Neste caso, o link leva o usuário ao site "https://www.exemplo.com".

Listas no HTML

HTML tem dois tipos de listas:

  • Lista não ordenada (com marcadores):
<ul>
<li>Primeiro item</li>
<li>Segundo item</li>
</ul>

Este código cria uma lista com marcadores.

  • Lista ordenada (numerada):
<ol>
<li>Primeiro item</li>
<li>Segundo item</li>
</ol>

Este código cria uma lista numerada.
